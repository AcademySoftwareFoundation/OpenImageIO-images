### `targa/

Most of these files were originally part Truevision's "TGAUTILS" distribution
containing the Targa file specification, code utilities, and these sample
images. It was available to all, free of charge, from their public FTP server:
ftp://ftp.truevision.com/pub/TGA.File.Format.Spec/PC.Version/
That company hasn't existed for a long time.

These are also test pattern images devoid of expressive content.

round_grill.tga is a regression test supplied by Doug Rogers, provided
by https://github.com/OpenImageIO/oiio/issues/2900
