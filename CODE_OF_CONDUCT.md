# Code of Conduct

The OpenImageIO project abides by the [Linux Foundation's code of conduct](https://lfprojects.org/policies/code-of-conduct).

To report incidents or to appeal reports of incidents, send email to
the Manager of LF Projects at manager@lfprojects.org.
