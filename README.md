This is a companion repository to the OpenImageIO source code
(https://github.com/AcademySoftwareFoundation/OpenImageIO), used just for
testing.

AcademySoftwareFoundation/OpenImageIO is for the source code to OpenImageIO.

AcademySoftwareFoundation/OpenImageIO-images is a place to store (potentially
large) test images and the like that we don't feel belong in the main code
repo.

🚑 Contact & reporting problems
-------------------------------

All communication about this repo is done via the main communications channels
of the OpenImageIO project:
- [GitHub project page](http://github.com/AcademySoftwareFoundation/OpenImageIO)
- [oiio-dev developer mail list](https://lists.aswf.io/g/oiio-dev) or [ASWF Slack](https://slack.aswf.io) (`#openimageio` channel) for general communication or requests for help.
- [OpenImageIO issues](https://github.com/AcademySoftwareFoundation/OpenImageIO/issues) to report bugs or make feature suggestions.
- [OpenImageIO security advisory](https://github.com/AcademySoftwareFoundation/OpenImageIO/security/advisories/new) for reporting true security vulnerabilities.
